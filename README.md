# WECC Grid Model

Western Electricity Coordinating Council (WECC) grid model based on GridPath (`https://github.com/blue-marble/gridpath`).

**Capabilities**:
* Capacity Expansion Model (CEM).
* Production Cost Model (PCM).

**Spatial resolution**:
* WECC Balancing Authorites (BA).

**Temporal resolution**:
* 4 investment periods (2025, 2030, 2035, 2040)
* CEM representative timepoints per period.
* PCM 8760 timepoints per period.

Industrial Decarbonization Grid team, the 2035 Initiative.
